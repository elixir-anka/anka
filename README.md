**`Installation`**


The package is [available in Hex](https://hex.pm/packages/anka), it can be installed
by adding `:anka` to your list of dependencies in `mix.exs`:

```elixir
def deps() do
  [
    {:anka, "~> 0.1.1"}
  ]
end
```
